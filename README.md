# Retail Sales Performance Analysis (Tableau)

## 📊 Project Overview
This project is a comprehensive analysis of retail sales and profitability from 2023 to 2025. Using Tableau, I transformed raw sales data into an interactive dashboard that identifies key growth drivers, regional dominance, and product-level profit margins.

## 🖥️ Final Dashboard
![Full Dashboard](images/Dashboard%20.png)

## 💡 Data Insights

* **Total Revenue:** The business generated **$350,500** in total sales.
* **Profitability Leader:** The **Electronics** category maintains the highest profit margins, even when sales volume fluctuates.
* **Regional Market Share:** The **East** and **West** regions are the primary revenue drivers.
* **Growth Trends:** Significant sales spikes were observed in **2024**, particularly within the Phones and Chairs product lines.
